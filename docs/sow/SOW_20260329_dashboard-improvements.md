## 1. エグゼクティブ・サマリー (Executive Summary)

ダッシュボードに「業務委託費分析」タブを新規追加（メイン）。業務分類別棒グラフ・メンバー別ドリルダウン・ピボットテーブルを実装し、委託費の分類・担当者別の可視化を実現した。あわせて、Google翻訳拡張による「直近→とりあえず」誤表示の根本対策（Dockerfileレベルで `translate="no"` を埋め込み）・タブ切り詰め修正・ページリネームによるナビゲーション安定化を実施し、Cloud Runへデプロイした。

---

## 2. プロジェクトの目的と背景 (Objectives & Background)

タダカヨでは約190名のメンバーが複数の業務分類（運営・事務・広報等）で業務委託として活動している。既存のスポンサー別タブでは業務分類ごとの委託費傾向を把握しにくく、担当者別のドリルダウン分析も不可能だった。本作業では業務委託費の分類別・メンバー別の可視化機能を新設し、コスト管理の透明性を向上させることを目的とした。また、Google翻訳拡張がStreamlitのUI文字列を誤訳するという慢性的なUX問題（「直近」→「とりあえず」等）の根本解消も合わせて実施した。

## 3. 実施内容詳細 (Technical Scope of Work)

### 3.1 業務委託費分析タブの新規追加

| 項目 | 内容 |
|------|------|
| 業務分類別棒グラフ | スポンサー別色分けの積み上げ棒グラフ。分類バーをクリックするとメンバー別ドリルダウンに切り替わる |
| ドリルダウン機能 | 選択分類のメンバー別内訳チャートを動的表示。メンバー数に応じて高さを自動調整（max(500, N×20+80)） |
| 凡例全表示 | `symbolLimit=0` を設定し、40名超の場合も凡例が切れないよう修正 |
| 分類一覧ピボットテーブル | 全行表示（`height=35×(N+1)+5`）。`reset_index()` でインデックス列のフォント色をデータ列と統一 |
| 操作ガイド表示 | 「クリック→ドリルダウン」「ダブルクリック→解除」をキャプションで案内 |
| リセット・選択解除ボタン | 「チャートをリセット」「選択解除」ボタンを追加 |

### 3.2 ページリネーム（pages/ → _pages/）

| 項目 | 内容 |
|------|------|
| ディレクトリリネーム | `pages/` → `_pages/` に変更。Streamlitの自動ページ検出を無効化し、ナビゲーションのフラッシュ（一瞬の点滅）を解消 |
| app.py参照更新 | 全ページのパスを `_pages/...` に更新 |

### 3.3 タブ切り詰め修正

| 項目 | 内容 |
|------|------|
| 内側タブ名を短縮 | 「月次推移」が「月次」に切れていた問題を修正。タブ名を短縮（例: メンバー別 月次支払額 → 月次支払額） |
| CSSタブパディング調整 | padding を `1.5rem → 0.75rem` に削減し、5タブが収まるよう対応 |

### 3.4 Google翻訳による誤表示の根本対策

| 項目 | 内容 |
|------|------|
| 原因特定 | ブラウザのGoogle翻訳拡張が「直近」→「とりあえず」、「月次推移」等を誤訳していた |
| Dockerfile修正 | `pip install` 後にStreamlitのHTMLテンプレート（`index.html`）を `sed` で直接書き換え。`<html translate="no" lang="ja">` を埋め込み、ページ受信時点から翻訳を根本的に無効化 |
