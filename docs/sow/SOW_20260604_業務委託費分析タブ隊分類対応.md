## 1. エグゼクティブ・サマリー (Executive Summary)

業務分類マスタの隊別分類への刷新に伴い、ダッシュボードの「業務委託費分析」タブ（Tab5）の集計方法を更新した。2026年5月以降のデータは新しい隊名（activity_category）で直接集計し、4月以前のデータは旧グループ名を隊名に正規化することで、月次推移グラフで前後を連続して比較できるようにした。移行期（5月中に切り替え進行中のレコード）はフォールバック処理で旧マップを経由して正規化する。

---

## 2. プロジェクトの目的と背景 (Objectives & Background)

### 2.1 背景
業務分類マスタが「タダカヨ業務全般」等の汎用分類から15隊別分類へ移行されたことで、ダッシュボードの業務委託費分析タブに旧グループ名と新隊名が混在する問題が発生した。

### 2.2 目的
- 旧データと新データを同じ隊名で統合表示し、継続的な推移分析を可能にする
- 移行期（2026年5月）の未整備レコードを旧マップで自動フォールバックする
- 凡例を新隊名のみに整理し、視認性を向上させる

## 3. 実施内容詳細 (Technical Scope of Work)

### 3.1 集計ロジック変更（PR #165）

| 項目 | 内容 |
|------|------|
| 2026年5月以降 | activity_category（隊名）を直接使用 |
| 2026年4月以前 | _LEGACY_GROUP_TO_TAI マップで旧グループ名を隊名に正規化 |
| 正規化マップ | 全旧グループ名→隊名 15パターンを定義 |

### 3.2 マッピング修正・凡例改善（PR #166）

| 項目 | 変更内容 |
|------|---------|
| タダレク | 旧: 色んな企業とwin-winになり隊 → 新: シン・もっと寄付を集め隊 |
| 行政事業（神奈川DX） | 旧: そのまま → 新: 神奈川県事業 |
| _COST_COLOR_DOMAIN | 旧グループ名を除去し新隊名のみに変更（凡例の不要項目を削除） |
| 非営利タブ除外 | 「神奈川県事業」を _COST_GROUP_EXCLUDE_NONPROFIT に追加 |
| 新規4隊追加 | それいけAI探検隊 / 個人情報をしっかり守り隊 / 一人ひとりを大切にし隊 / 介護DXで包括の未来を応援し隊 |

### 3.3 移行期フォールバック対応（PR #167）

| 項目 | 内容 |
|------|------|
| 課題 | 2026年5月データで activity_category が旧値（業務分類名等）のレコードが残存 |
| 対応 | _VALID_TAI_NAMES セット（有効な隊名一覧）を追加し、activity_category がこのセットに含まれない場合は旧マップ経由で正規化するフォールバックを実装 |
