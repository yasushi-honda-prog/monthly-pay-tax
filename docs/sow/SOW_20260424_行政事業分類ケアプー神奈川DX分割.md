## 1. エグゼクティブ・サマリー (Executive Summary)

業務委託費分析タブの「行政事業」分類を、スポンサーフィールドを基にケアプーと神奈川DXの2分類に分割した。統括隊長会議での管轄変更（神奈川DXプロジェクトの管轄移管）を受けた対応で、各グループの費用を個別に可視化できるようになった。

---

## 2. プロジェクトの目的と背景 (Objectives & Background)

### 2.1 背景
直近の統括隊長会議で神奈川DXプロジェクトの管轄が変更となった。これにより、これまで一括表示していた「行政事業（ケアプランデータ連携＆神奈川県事業）」分類では両プロジェクトの費用が混在し、各管轄での集計・確認が困難な状態となっていた。

### 2.2 目的
- 神奈川DX（スポンサー：神奈川県DX）とケアプー（スポンサー：ケアプー事業（全国統一））の業務委託費をグラフ上で別分類として表示する
- スポンサーフィールドによる自動判定とし、識別不可能な行（移動手当・電話応対等）はすべてケアプーに集計する

---

## 3. 実施内容詳細 (Technical Scope of Work)

### 3.1 分類マッピング変更（_COST_GROUP_MAP）

| 項目 | 内容 |
|------|------|
| 変更前 | 行政事業9業務分類すべてを「行政事業（行政事業：ケアプランデータ連携システムを広め隊＆神奈川県事業）」に統合 |
| 変更後 | 同9業務分類のデフォルト値を「行政事業（ケアプー：ケアプランデータ連携システムを広め隊）」に変更 |
| 振り替えロジック | 業務分類マップ適用後、`sponsor == "神奈川県DX"` かつ `cost_group == "行政事業（ケアプー：...）"` の行を「行政事業（神奈川DX）」に上書き |
| 対象外 | スポンサー対応・タダスク等の他分類は変更なし |

### 3.2 カラードメイン・非営利活動除外設定の更新

| 項目 | 内容 |
|------|------|
| _COST_COLOR_DOMAIN | 「行政事業（神奈川DX）」を色ドメインに追加（グラフ凡例に固定色で表示） |
| _COST_GROUP_EXCLUDE_NONPROFIT | 旧ラベルを削除し、ケアプー・神奈川DXの2ラベルを追加（非営利活動タブでは両方除外） |

## 5. 品質保証と受入基準 (Quality Assurance & Acceptance)

- `sponsor == "神奈川県DX"` の行は「行政事業（神奈川DX）」に分類されること
- `sponsor != "神奈川県DX"` の行（ケアプー事業・移動手当・電話応対等）は「行政事業（ケアプー：ケアプランデータ連携システムを広め隊）」に分類されること
- スポンサー対応・タダスク等の他分類が影響を受けないこと
- 非営利活動タブでは神奈川DX・ケアプー両分類とも除外されること
- グラフの凡例に両分類が固定色で表示されること
