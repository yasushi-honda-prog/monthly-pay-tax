## 1. エグゼクティブ・サマリー (Executive Summary)

業務委託費分析タブの「行政事業」分類をケアプーと神奈川DXの2分類に分割し、スポンサー未入力行の内容欄キーワード補完・令和8年度新業務分類の追加・ドリルダウン用セレクトボックスの追加・コンソール警告の解消を行い、分析精度とUXを改善した。

---

## 2. プロジェクトの目的と背景 (Objectives & Background)

### 2.1 背景
直近の統括隊長会議で神奈川DXプロジェクトの管轄が変更となった。これにより、これまで一括表示していた「行政事業」分類では両プロジェクトの費用が混在し、各管轄での集計・確認が困難な状態となっていた。また、スポンサー未入力のメンバーが多く正確な分類が困難であること、棒グラフが小さい場合のドリルダウン操作しにくさも課題となっていた。

### 2.2 目的
- 神奈川DX（スポンサー：神奈川県DX）とケアプー（スポンサー：ケアプー事業（全国統一））の業務委託費をグラフ上で別分類として表示する
- スポンサー未入力行は内容欄キーワードで神奈川DXを補完し、分析精度を高める
- 令和8年度行政事業（各事業のPM・AM）を分類マップに追加する
- バーが小さくて選択しにくい場合のセレクトボックス代替手段を提供する
- Vega-Lite コンソール警告を解消する

---

## 3. 実施内容詳細 (Technical Scope of Work)

### 3.1 分類マッピング変更（_COST_GROUP_MAP）

| 項目 | 内容 |
|------|------|
| 変更前 | 行政事業分類すべてを「行政事業（ケアプランデータ連携＆神奈川県事業）」に統合 |
| 変更後 | デフォルトを「行政事業（ケアプー：ケアプランデータ連携システムを広め隊）」に変更 |
| 振り替えロジック① | `sponsor == "神奈川県DX"` かつ対象分類の行を「行政事業（神奈川DX）」に上書き（スポンサー対応2分類・エリアリーダー分類含む） |
| 振り替えロジック② | スポンサー未入力補完：`description` に「神奈川DX」「神奈川県DX」「神奈川県」を含む行を「行政事業（神奈川DX）」に振り替え |
| 令和8年度追加 | 令和8年度行政事業（各事業のPM・AM）を `_COST_GROUP_MAP` に新規追加 |

### 3.2 カラードメイン・非営利活動除外設定の更新

| 項目 | 内容 |
|------|------|
| _COST_COLOR_DOMAIN | 「行政事業（神奈川DX）」を色ドメインに追加（グラフ凡例に固定色で表示） |
| _COST_GROUP_EXCLUDE_NONPROFIT | 旧ラベルを削除し、ケアプー・神奈川DXの2ラベルを追加（非営利活動タブでは両方除外） |

### 3.3 ドリルダウン用セレクトボックス追加

| 項目 | 内容 |
|------|------|
| 追加UI | チャート下部にドロップダウンを追加。バーが小さくて選択しにくい分類（神奈川DX等）をドロップダウンからも選択可能 |
| 既存動作 | バークリックによるドリルダウンはそのまま維持 |
| 選択解除 | 「選択解除」ボタン・「チャートをリセット」ボタンでドロップダウンも同時リセット |

### 3.4 Vega-Lite コンソール警告対応

| 項目 | 内容 |
|------|------|
| 警告内容 | `Infinite extent for field "合計": [Infinity, -Infinity]` 等 9件 |
| 原因 | `total_hover` チャートの Y エンコーディングに `stack=False` が未設定で、Vega-Lite v6 がスタック計算を試みていた |
| 修正 | `y=alt.Y("合計:Q", stack=False)` を明示設定 |
| 結果 | エラー 0件・警告 0件に改善 |

## 5. 品質保証と受入基準 (Quality Assurance & Acceptance)

- `sponsor == "神奈川県DX"` の行は「行政事業（神奈川DX）」に分類されること
- スポンサー未入力でも内容欄に「神奈川DX」「神奈川県DX」「神奈川県」を含む行が「行政事業（神奈川DX）」に分類されること
- 令和8年度行政事業（各事業のPM・AM）がスポンサーに応じて正しく振り分けられること
- ドロップダウンで分類を選択するとドリルダウン内訳が表示されること
- 「選択解除」「チャートをリセット」でドロップダウンがリセットされること
- ブラウザコンソールにエラー・警告が出ないこと
- 非営利活動タブでは神奈川DX・ケアプー両分類とも除外されること
